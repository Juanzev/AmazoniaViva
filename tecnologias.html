Stack Tecnológica

As ferramentas que dão vida ao sistema de monitoramento.

Python & Flask

Utilizamos o Python no backend pela sua robustez e facilidade de integração com hardware. O framework Flask gerencia as rotas e a comunicação em tempo real com o frontend.

C++ (Arduino)

O coração do hardware. O código em C++ processa os sinais analógicos dos sensores de fumaça e envia os alertas via porta Serial de forma extremamente rápida.

Tailwind CSS

Para uma interface moderna e responsiva. O Tailwind nos permite criar um design personalizado em tons de verde com performance otimizada e visual profissional.

HTML5 Semantic

Estruturação de dados limpa e acessível. Garantimos que o sistema seja otimizado para SEO e fácil de manter com as melhores práticas de marcação.

JavaScript ES6

Responsável pelo dinamismo da página. O JS realiza o polling assíncrono para buscar o status do sensor sem a necessidade de recarregar o navegador.

Versionamento

Controle total do desenvolvimento. Utilizamos Git para garantir a integridade do código e facilitar a colaboração em equipe no projeto.
