Sobre o Sistema

Missão

O Amazonia Viva foi desenvolvido com o objetivo de mitigar os danos causados por queimadas através da detecção precoce de fumaça e calor. Utilizando sensores de baixo custo e alta precisão, o sistema permite uma resposta rápida das autoridades competentes.

Funcionamento

O hardware (Arduino) monitora constantemente a qualidade do ar. Ao detectar níveis de fumaça acima do limite seguro, um sinal é enviado via comunicação Serial para o servidor Flask, que atualiza a interface web instantaneamente para todos os usuários conectados.

Arquitetura do Sistema

1

Coleta de Dados

Sensores analógicos MQ-2 processados por microcontrolador ATmega328P.

2

Processamento Local

Algoritmo em C++ que filtra ruídos e valida o limite de detecção (Threshold).

3

Interface Web

Servidor Python/Flask integrando o hardware ao usuário final via WebSockets/Polling.
